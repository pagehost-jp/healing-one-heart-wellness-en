### 📐 レイアウトの改善

#### デスクトップ固定幅レイアウト (style.css:193-203)
- **目的**: デスクトップでブラウザサイズを変更しても、テキストの改行位置が変わらないようにする
- **実装**: メディアクエリで768px以上のみ固定幅1200pxに設定

```css
/* デスクトップは固定幅 */
@media (min-width: 768px) {
    .container {
        width: 1200px;
        max-width: none;
    }
}
```

- **モバイル**: 元のまま `max-width: var(--container-width)` でレスポンシブ維持

### ⚠️ 試行錯誤した内容（最終的に取り消し）

#### 1. コース画像の縦位置調整
- **試行内容**: `object-position: center 85%` を追加して画像を下方向に移動
- **問題**: 画像が拡大されすぎて不自然になった
- **対応**: 全て元に戻し、デフォルトの `object-fit: cover` のみ使用

#### 2. 日本語テキストの改行制御
- **試行内容**: `word-break: keep-all` などのCSSプロパティで改行制御を試みた
- **問題**: テキストが横にオーバーフローして、受講生の声セクションがはみ出した
- **対応**: 全てのline-break関連CSSを削除し、元の状態に戻した
- **結論**: ブラウザのデフォルト改行動作に任せる
