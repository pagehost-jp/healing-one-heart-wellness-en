# 画像フォルダ

このフォルダに以下の画像を配置してください：

## プロフィールセクション用
- profile.jpg - プロフィール写真（推奨サイズ: 400x400px以上、正方形）
- badge-iin.png - IINバッジ（推奨サイズ: 240x120px程度）
- certificate.png - 認定証画像（推奨サイズ: 600x400px程度）

## オンラインコースセクション用
- courses-hero.jpg - セクションヒーロー画像（推奨サイズ: 1600x900px、16:9比率）
- course-1.jpg - コース1サムネイル（推奨サイズ: 600x400px）
- course-2.jpg - コース2サムネイル（推奨サイズ: 600x400px）
- course-3.jpg - コース3サムネイル（推奨サイズ: 600x400px）

## 注意事項
- 画像が配置されていない場合は、プレースホルダーが自動的に表示されます
- 画像形式: JPG, PNG推奨
- ファイル名は正確に一致させてください
