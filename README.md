# Healing One Heart LP (English Version)

Landing page for healing and coaching services - English version

## Project Overview

This is the English version of the Healing One Heart landing page, targeting women in their 50s seeking holistic wellness support.

## Target Audience

- Women in their 50s
- Those who have been caring for others
- Seeking balance in mind, body, and soul

## Design Concept

- **Clean white base**: Clarity and peace
- **Ample whitespace**: Comfortable reading experience
- **Soft pink accents**: Warmth and compassion
- **No hard selling**: Quiet trust and authenticity

## Page Structure

1. **Hero Section** - Resonating message without being too abstract
2. **Empathy Section** - Understanding the reader's feelings
3. **Philosophy** - Not "fixing" but "harmonizing"
4. **Services** - Three support offerings
5. **Story Section** - Background without being overly sentimental
6. **CTA** - Gentle invitation to connect
7. **Footer** - Simple and reassuring

## Tech Stack

- HTML5
- CSS3
- JavaScript (minimal)
- GitHub Pages ready

## Local Development

1. Open `index.html` in your browser
2. Or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
   → Access http://localhost:8000

## GitHub Pages Deployment

### 1. Create GitHub Repository

Create a new repository on GitHub.com:
- Repository name: `healing-one-heart-lp-en`
- Public setting

### 2. Push from Local

```bash
# Add remote repository
git remote add origin https://github.com/YOUR_USERNAME/healing-one-heart-lp-en.git

# Commit files
git add .
git commit -m "Initial commit: English version LP"

# Push to main branch
git branch -M main
git push -u origin main
```

### 3. Enable GitHub Pages

1. Go to repository Settings
2. Select "Pages" from left menu
3. Source: Select "Deploy from a branch"
4. Branch: Select "main" and "/ (root)", then "Save"
5. URL will be available in a few minutes:
   - `https://YOUR_USERNAME.github.io/healing-one-heart-lp-en/`
